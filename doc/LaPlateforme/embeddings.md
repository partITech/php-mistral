## Embeddings

Send an array of sentences to the Embedding model.
> [!TIP] 
> By default the 'mistral-embed' model is used. 
> But for any reason, if you need to change this, you simply need to pass the model name in a second parameter.
> 
> 
> 
> `$client->embeddings(datas: ["What is the best French cheese?"], model: 'my-model');`

```php

use Partitech\PhpMistral\Clients\Mistral\MistralClient;use Partitech\PhpMistral\MistralClientException;

$client = new MistralClient($apiKey);

try {
    $embeddingsBatchResponse = $client->embeddings(["What is the best French cheese?"]);
} catch (MistralClientException $e) {
    echo $e->getMessage();
    exit(1);
}

```
